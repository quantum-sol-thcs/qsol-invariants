## ✦ Purpose of This Directory

This `/docs` directory serves as the **institutional backbone** of the QSOL
Invariant Evolution Core. It ensures that:

- the invariant canon remains stable  
- governance is explicit and transparent  
- contributors understand the philosophy behind the system  
- maintainers have a clear, authoritative reference  
- the project remains audit‑ready and future‑proof  

All technical, architectural, and philosophical decisions must align with the
documents contained here.
